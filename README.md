# SimulatorFramework
Eeste framewrok foi desenvolvido por mim para a criação do meu Simuator tendo grande ajuda do [Rojo](https://github.com/rojo-rbx/rojo) 7.6.1.que integrava a IDE com o Studio. Também tive grande ajuda do [quenty](https://x.com/quenty) que disponibilizou uma biblioteca incrível que não coloquei aqui, mas deixei o link dela disponível [aqui](https://github.com/Quenty/NevermoreEngine)

## Como usar este Framework?
Baixe os scripts dentro da pasta src e e baixe a pluguin do do [Script Sync](https://devforum.roblox.com/t/studio-script-sync-now-in-beta/4065468) que ele irá depositar automaticamente na sua IDE integrada ao rojo


[the Rojo documentation](https://rojo.space/docs).
